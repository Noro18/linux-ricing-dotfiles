    <h2>• How Everything work together •</h2>
    <h3></h3>
</div>

<details> 
  <summary>Dynamic Colors</summary>

  The dynamic is generated by using `matugen` and `walllust` by the wallpaper. 

  Here's a more detailed explanation

### Selected Wallpaper

  1. You open a wallpaper selection menu using the command `wallset` (this is store in `~/.local/bin`)
  2. After you select a wallpaper a `wallset-backend` will run doing 
     1. `swww` change wallpaper
     2. `matugen` & `wallust` generate the color scheme
     3. And then wabar, kitty, hyprland, and rofi will use that color  
    
### Random Wallpaper

This is more straight  forward as it only run the script `wallset-backend-startup`.
</details>
