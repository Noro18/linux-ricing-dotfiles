# Kitty

A GPU-based terminal emulator that supports tiles, images, and true color.

---

## File Structure

```
~/.config/kitty/
├── kitty.conf                # Main configuration file
├── colors/                   # Additional color schemes (optional)
│   └── colors.conf           # Optional static colors (not used in this setup)
└── custom.conf               # Optional user overrides (not versioned)
```

---

## How the Files Relate

- `kitty.conf` is the main configuration file that sets font, window behavior, and includes the color configuration.
- The color configuration is pulled from `~/.cache/wallust/colors-kitty.conf`, which is generated by `wallust` on every wallpaper change.
- There is a commented line showing an alternative way to include colors (`include ./colors/colors.conf`), but the active method uses the wallust-generated file.

---

## How Colors Work

Kitty gets its colors from the `wallust` tool, which generates a 16-color terminal palette from the wallpaper.

| File | Generated by | When |
|------|-------------|------|
| `~/.cache/wallust/colors-kitty.conf` | wallust | On every wallpaper change |

This file contains definitions for `color0` through `color15`, `foreground`, and `background` that Kitty uses for:
- Text colors
- Background color
- Cursor color (if set to use foreground)
- Selection colors
- Any color escapes or themes used by terminal applications

To change how colors are applied to Kitty, edit the `wallust` template:
- `~/.config/wallust/templates/kitty.conf`

This template defines how the wallust-generated color palette is formatted for Kitty.

---

## Key Configuration Options in kitty.conf

- `font_family`: Set to "JetBrainsMono Nerd Font" for icons and programming ligatures.
- `font_size`: Base font size (11pt).
- `window_padding_width`: Padding inside the window (20 pixels).
- `hide_window_decorations`: Hides the title bar for a cleaner look (enabled).
- `background_opacity`: Sets the background to 80% opaque for a slight transparency effect.
- `scrollback_lines`: Keeps 2000 lines of history.
- `cursor_blink_interval`: Cursor blinks every 0.5 seconds.
- `cursor_stop_blinking_after`: Cursor stops blinking after 1 second of inactivity.
- `selection_foreground` and `selection_background`: Set to "none" to use the terminal's default selection colors (which come from the theme).

## ⚠️ Things Not to Edit Manually

- `~/.cache/wallust/colors-kitty.conf` — overwritten by wallust on wallpaper change
