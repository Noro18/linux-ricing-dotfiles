## How the Files Relate

Rofi is launched via scripts in the `launchers/type-*/launcher.sh` files. These scripts call `rofi` with a specific theme file (`style-*.rasi`) and may include applets.

The theme files (`.rasi`) define the visual appearance of Rofi. They import colors from either:
- `colors-matugen.rasi` (for Material You colors from matugen)
- `colors-wallust.rasi` (for terminal palette colors from wallust)

These color files are generated automatically when the wallpaper changes.

---

## How Colors Work

| File | Generated by | When |
|------|-------------|------|
| `colors-matugen.rasi` | matugen | On every wallpaper change |
| `colors-wallust.rasi` | wallust | On every wallpaper change |

To change how colors map to Rofi elements, edit the generator templates:
- `~/.config/matugen/templates/` for Material You colors (look for `.rasi` templates)
- `~/.config/wallust/templates/` for terminal palette colors (look for `.rasi` templates)

---

## Changing Launchers and Applets

1. **To switch launcher types**: Run the appropriate `launcher.sh` script (e.g., `~/.config/rofi/launchers/type-1/launcher.sh`). Each type offers different layouts and features.
2. **To customize a launcher**: Edit the `.rasi` files in the launcher's style directory (e.g., `~/.config/rofi/launchers/type-1/style-1.rasi`).
3. **To add/modify applets**: 
   - Modify the scripts in `~/.config/rofi/applets/bin/` for functionality
   - Edit the `.rasi` files in `~/.config/rofi/applets/type-*/` for appearance
   - Shared styles are in `~/.config/rofi/applets/shared/`
